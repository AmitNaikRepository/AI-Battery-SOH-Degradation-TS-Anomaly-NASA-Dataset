üéØ Project Overview

This project builds a machine learning model to predict battery capacity degradation using NASA battery dataset. The model achieves 88.4% R¬≤ accuracy on unseen batteries with zero data leakage.

Key Achievement: Production-ready XGBoost model that predicts remaining battery capacity based on operational measurements, enabling proactive maintenance and warranty cost prediction.

üìä Dataset

  • Source: NASA Battery Dataset
  • Batteries Used: 4 batteries (B0005, B0006, B0007, B0018)
  • Total Cycles: ~600 discharge cycles
  • Features: 18 engineered features from 5 raw measurements
  • Target: Battery capacity (Ah)

Train/Test Split Strategy

  • Training: 3 batteries (B0005, B0006, B0007) - 489 samples
  • Testing: 1 unseen battery (B0018) - 127 samples
  • Strategy: Battery-level split (not random) to test true generalization

üö´ Data Leakage Prevention

We identified and removed 13 features that caused data leakage - features that had suspiciously high correlation with the target or were directly calculated from capacity.

Figure 3: Before vs After - Shows removed leakage features vs clean features kept

Excluded Features (Data Leakage):
  • capacity_fade_total - Direct calculation from capacity
  • capacity_fade_percent - Direct calculation from capacity
  • discharge_capacity_ratio - = Capacity / initial_capacity
  • capacity_rolling_mean_5 - ‚âà Capacity itself (r=0.996)
  • energy_discharged - = Capacity √ó Voltage (r=0.999)
  • And 8 more similar features...

Correlation Analysis

  • Before removal: Some features had correlation > 0.99 with target
  • After removal: All features have correlation < 0.90
  • Result: Model performance dropped from 99.7% R¬≤ (with leakage) to 88.4% R¬≤ (realistic)
